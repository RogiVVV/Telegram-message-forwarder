## Для запуска бота:

1) Клонируйте репозиторий
2) Создайте рядом с файлами исходного кода файл config.py
3) Укажите в нём токен Telegram-бота:

``` python
TOKEN = "ваш_токен_бота"
```

Теперь можете запускать бота

---

## Команды

`/add_source_ids` - добавляет чаты в список ресурсов. После этого сообщения, присланные в выбранные чаты, будут копироваться в ваш \
`/remove_source_ids` - убирает чаты из списка ресурсов \
`/set_allowed_types` - настраивает типы пересылаемых ботом сообщений. Бот будет будет пересылать только сообщения выбранного типа \
`/check_sources` - показывает список всех ресурсов

Учтите: бот не будет пересылать сообщения без `/add_source_ids`
