## Для запуска бота:

1) Клонируйте репозиторий
2) Создайте рядом с файлами исходного кода файл config.py
3) Укажите в нём токен Telegram-бота:

``` python
TOKEN = "ваш_токен_бота"
```

Теперь можете запускать бота

---

## Команды

`/add_target_id` - добавляет чат в список адресатов. Теперь сообщения, пересланные в этот чат будут копироваться в выбранный \
`/remove_target_id` - убирает чат из списка адресатов \
`/set_source_target_ids` - настраивает пересылку из других чатов. В основном для групп и каналов. Например, вы добавили бот в телеграм канал и хотите, чтобы сообщения оттуда копировались в вашу группу с друзьями \
`/set_allowed_types` - настраивает типы пересылаемых ботом сообщения. Бот будет будет пересылать только сообщения выбранного типа
